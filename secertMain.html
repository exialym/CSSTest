Similarly, a darker shadow at the bottom (or a lighter one at the top), creates the illusion that an object is extruded from the main surface. The reason this works is that we usu- ally assume that the light source is above us, so an extruded object would create a shadow underneath it, and an embossed object would be lit at the bottom.
